Edition de code

Les éditeurs de code sont des logiciels conçus afin de simplifier l’écriture de code. À l'aide de raccourcis ou d’éléments visuels ils permettent une plus grande rapidité et une plus grande facilité dans la conception de site internet. L’éditeur de code est un outil fondamental pour les métiers du web. Il existe un grand nombre d’éditeurs de code, ayant chacun ses particularités.
